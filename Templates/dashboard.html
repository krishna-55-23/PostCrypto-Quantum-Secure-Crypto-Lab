Supported Cryptographic Algorithms

⚙️ Classical Algorithms
Classical

RSA

Public-key cryptography based on large prime factorization. Supports 1024–4096 bit keys.

Classical

ECC

Elliptic curve cryptography with smaller keys and equivalent security to RSA.

Symmetric

AES

Symmetric block cipher. Supports 128, 192, and 256-bit secret keys via EAX mode.

🔬 Post-Quantum Algorithms
Post-Quantum KEM

Kyber

NIST-standardized key encapsulation mechanism designed for quantum resistance.

Post-Quantum Signature

Dilithium

Lattice-based digital signature algorithm providing secure quantum-resistant authentication.
